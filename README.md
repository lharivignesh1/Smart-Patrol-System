# 🚔 Smart Patrolling System

> *"Innovation isn't about having unlimited resources—it's about making the most of the ones you have."*

## 📖 The Story Behind This Project

This project was developed during the **Bid & Build** event organized by the **IEEE Club**.

Unlike a traditional hackathon, the challenge began long before writing a single line of code.

Every team was given a limited number of virtual credits and had to strategically **bid for APIs and datasets**. Whatever resources we managed to win would be the only ones available to build a complete solution within **3 hours**.

After an intense bidding session, our team secured the **Crime City Dataset** by investing **350 credits**.

Although this meant we couldn't acquire additional APIs or datasets, we accepted the challenge and decided to build an impactful solution using only a single dataset.

That decision led to the creation of the **Smart Patrolling System**.

---

# 🎯 Problem Statement

Traditional police patrolling often follows fixed schedules or manual decisions, making it difficult to identify and monitor high-risk locations efficiently.

At the same time, citizens usually have no easy way to understand the safety level of an area before visiting it.

There is a need for a smarter, data-driven approach that can assist both law enforcement agencies and the public.

---

# 💡 Our Solution

The **Smart Patrolling System** is an AI-powered web application that analyzes city-level data to estimate crime risk percentages and helps optimize patrol deployment.

The system provides valuable insights for both citizens and police departments, enabling informed decisions based on AI-generated risk analysis.

---

# ✨ Features

### 👤 Public Portal

* View city-wise crime risk percentage
* Check the safety level before travelling
* Simple and user-friendly interface
* AI-generated risk analysis

### 🚔 Police Command Center

* Monitor high-risk cities
* View Top Risk Zones
* Deploy patrol units
* Automatic emergency area prioritization
* Dynamic patrol simulation
* Risk reduction after successful patrol deployment

---

# ⚙️ How It Works

1. Load the Train City Dataset.
2. Analyze the available city data.
3. Generate AI-based crime risk scores.
4. Convert scores into understandable percentages.
5. Rank cities according to risk level.
6. Display high-risk locations on the dashboard.
7. Allow police officers to deploy patrol units.
8. Simulate reduced crime risk after patrol deployment.
